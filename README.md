## Stack

- Vue 3 (with [Composition API](https://v3.vuejs.org/guide/composition-api-introduction.html))
- ESlint
- Prettier
- Typescript
- [Vite](https://vitejs.dev/guide/)
- Jest
- Tailwind
- Github Actions (CI/CD)
